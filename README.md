# PCA Assignment – Advanced Linear Algebra

##  Project Overview
This project implements Principal Component Analysis (PCA) from scratch using NumPy. The goal is to reduce the dimensionality of a real-world African socioeconomic dataset while preserving variance.

---

##  Dataset
- Source: African socioeconomic dataset
- Features: Economic activity, population pressure, and related indicators
- Contains missing values handled using imputation
- Includes both numeric and categorical variables

---

##  Methods Used
- Data cleaning and preprocessing
- Handling missing values
- Encoding categorical variables
- Covariance matrix computation
- Eigenvalue and eigenvector decomposition
- PCA transformation
- Explained variance analysis

---

##  Results
- Reduced dimensionality while preserving over 91% variance
- First few principal components captured major socioeconomic patterns
- Significant reduction in redundancy between variables
